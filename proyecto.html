Inicio

Este proyecto permite analizar distintos escenarios relacionados con el abastecimiento de carburantes, costos de transporte y compras familiares mediante simuladores interactivos desarrollados con HTML, CSS y JavaScript.

Contexto

En Bolivia diversas situaciones económicas pueden afectar el abastecimiento de combustibles, incrementar costos de transporte y modificar el presupuesto de las familias.

Estos simuladores ayudan a comprender el impacto de dichos factores mediante cálculos automáticos.

Conclusiones

Los simuladores muestran cómo factores económicos pueden afectar a la población y permiten analizar escenarios de forma práctica e interactiva.
