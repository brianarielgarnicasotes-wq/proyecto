# proyecto
Simulador Web de Impacto Económico
Descripción
Este proyecto fue desarrollado para la materia de Programación Web I. Consiste en una página web con simuladores que permiten analizar situaciones relacionadas con el abastecimiento de carburantes, el transporte y las compras familiares.
Tecnologías utilizadas
HTML
CSS
JavaScript
Funciones
Simulador de abastecimiento de carburantes.
Simulador de transporte.
Simulador de compras familiares.
En estos simuladores nos ayudaran en especial en el calculo mostrando el resultado con un fondo variado en tres:
Normal:Verde
Alerta:Amarillo
Critico:Rojo
Facilitando la compresion del resultado
